# Story Mapper for Jira — Support

This is the public support tracker for **Story Mapper for Jira**, available on the [Atlassian Marketplace](#).

## Reporting a bug

Open an [Issue](../../issues/new?template=bug_report.md) and select **Bug report**. Include:
- What you expected vs. what happened
- Your Jira Cloud site URL (e.g. `yourcompany.atlassian.net`)
- Steps to reproduce

## Requesting a feature

Open an [Issue](../../issues/new?template=feature_request.md) and select **Feature request**.

## Other questions

Email: support@dtaltechnologies.com

We're a small team. We aim to respond to bugs within 3 business days and feature requests within 7.
